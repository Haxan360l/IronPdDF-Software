# IronPdDF-Software Front-End Test
Implemntation of IronPDF website based on a provided Figma design.
It is developed using CodeIgniter 4 engine, Bootstrap, custom CSS, and a JSON database with a little bit of Javascript.
Below are the materials/ tools used;
CodeIgniter 4
Bootstrap 5
Custom CSS
PHP (views)
JSON for data

Structure
codeIgnniter4/app/Views - Home Page and sections ie Navigation, footer etc
/public/assets/ - CSS, JavaScript and Images.
data/page_content.json - Navigation and content configurations.

Features 
Figma pixel-aligned layout
Json driven assets 
Optimised images as in the figma design
Interactive navigation
Responsive design accross desktop and mobile

Browser Support
The website supports all browsers as tested on;
Google Chrome
Brave
Mozilla Firefox
Microsoft Edge

Responsiveness
It adapts across common breakpoints;
Desktop > 1200px
Laptop (992px - 1199px)
Tablet (768px - 991px)
Mobile (576px)

SEO and perfomance
Semantic HTML
Proper heading hierarchy
Room for use of Alt text
Core web vitals considered LCP, CLS, INP

QA and Validation
Performed a full QA checklist on;
Pixel Accuracy compared to the Figma design
Typography
Cross-browser compatibility
SEO validation
Responsiveness
Perfomance
